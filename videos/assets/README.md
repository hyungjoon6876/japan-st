# Video mascot assets

These four PNG poses come from the user-provided `GLOAT x GILT Design.zip` design handoff:

- `mascot-nyamnyam.png`
- `mascot-dance.png`
- `mascot-cheer.png`
- `mascot-giggle.png`

They are used as the animated reaction character in the Japanese lesson videos. The video generator keeps the original PNGs unchanged and applies motion while encoding.

The matching `*-sticker.png` files only add an alpha-channel rounded-corner mask so the pale source backgrounds sit cleanly on the video canvas.
