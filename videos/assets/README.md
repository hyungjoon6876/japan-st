# Video mascot assets

These nine PNG poses come from the user-provided `GLOAT x GILT Design.zip` design handoff:

- `mascot-nyamnyam.png`
- `mascot-dance.png`
- `mascot-cheer.png`
- `mascot-giggle.png`
- `villain-1-peek.png`
- `villain-2-hack.png`
- `villain-3-steal.png`
- `villain-4-hug.png`
- `og-mascot.png`

They are the on-screen lead character in the 11-part Yomi Jjapjjap story. The generator keeps the source PNGs unchanged, creates temporary rounded masks, and applies entrance, bounce, peek, hop, and dance motion while encoding.

The four matching `*-sticker.png` files from the first version only add an alpha-channel rounded-corner mask. New builds prepare equivalent temporary masks under `videos/.build/mascots/`.
