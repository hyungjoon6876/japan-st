# 初級日本語 — 일본어 학습 도구

스캔한 일본어 초급 문법 교재 한 권을 통째로 옮기고, 한자 데이터(교육·상용 2,136자)를 병합한
**단일 파일 정적(static) HTML** 학습 앱입니다. 서버·인터넷 없이 `index.html`을 브라우저로 열면 동작합니다.

## 담긴 것
- **문형** 115개 · **낱말** 824개 · **예문** 517개(읽기·한국어 번역) · **활용표** 70개 · **연습문제** 353문항(정답)
- **초보 가이드** 23주제(원리·규칙표·✕→○ 실수·암기 팁)
- **한자** 2,136자(뜻·음훈독·한자음·JLPT·학년) + **문맥 연결 렌즈**(한자 ↔ 교재 낱말·예문)
- **교재 커버리지 맵**(학년/JLPT 대비 교재가 다루는 한자)

## 기능
- 통합 SRS(Leitner) 퀴즈·플래시카드, 오답노트, 진도 저장(localStorage)
- 후리가나 3단 토글 · 몰입모드(한국어 숨김) · 다크/라이트 · 검색
- 완전 오프라인(외부 요청 0), 라이트·다크 대응

## 사용
`index.html` 을 브라우저로 열기만 하면 됩니다.

## 🎧 듣기 트랙 (핸드폰용)

`audio/` 폴더에 **레슨별 MP3 학습 트랙**(11개, 총 ~42MB)이 있습니다. 각 트랙은
**일본어(읽기) → 한국어 뜻/번역** 순으로 낱말과 예문을 읽어줍니다. 핸드폰에 넣고
**기본 음악앱**으로 재생하면 잠금화면·백그라운드·이어폰 컨트롤이 그대로 동작해
**걸으면서 들을 수** 있습니다.

- 음성: Microsoft 신경망 TTS (일본어 ja-JP-Nanami, 한국어 ko-KR-SunHi)
- 트랙 순서 = 교재 레슨 순서 (앨범: `初級日本語 · 듣기 학습`)
- HTML 앱 자체에도 **「🔊 소리」** 모드가 있어, 화면의 일본어를 탭하면 그 자리에서 발음을 들려줍니다(화면 켠 상태).
